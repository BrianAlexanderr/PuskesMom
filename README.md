**PuskesMom** adalah aplikasi Android berbasis Java yang dirancang untuk membantu ibu memantau tumbuh kembang anak secara digital. Aplikasi ini mengintegrasikan layanan informasi kesehatan, jadwal imunisasi, dan konsultasi medis dalam satu platform.

## âœ¨ Fitur Utama

* **Monitoring Tumbuh Kembang**: Pencatatan data fisik anak (berat badan, tinggi badan, usia).
* **Prediksi Stunting**: Fitur deteksi dini risiko stunting menggunakan analisis data.
* **Jadwal Imunisasi**: Kalender otomatis untuk memantau jadwal vaksinasi anak.
* **Konsultasi Dokter (Chat)**: Fitur pesan instan untuk berkonsultasi langsung dengan pakar kesehatan.
* **Manajemen Profil Anak**: Kemudahan mengelola lebih dari satu profil anak dalam satu akun.
